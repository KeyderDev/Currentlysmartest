- 1. ¿Qué es un ataque DDOS?

Todos los servidores tienen un limite de usuarios. Servidores de Juegos, Paginas WEB, y telefonia

pueden soportar un ancho de banda especifico. Una vez explicado esto, ¿Qué es un ataque DDOS?


Los ataques de denegacion de servicio distribuida, o mejor conocidos como (DDOS), son ataques que se hacen a un servidor

con la intencion de denegar el servicio, y que nadie se pueda conectar al servidor.


Estos ataques surgen porque mucha gente a la vez se esta conectando al servidor, y al servidor no tener los recursos suficientes,

(cpu y ram), pues el servidor se cae.


Si queremos denegar el servicio de un servidor, como conseguimos tanta gente para que se conecta a la vez a un mismo servidor?

Para eso existen las Botnet



- 2. ¿Qué es una Botnet?

Las Botnet son redes que se usan para denegar el servicio a servidores. En el negocio de la Deep Web, se venden Botnets enteras.


Asi funciona una Botnet, el atacante va buscando dispositivos en el Iot (Internet of things) a los que conectarse, como Camaras CCTV, Routers, DVR'S entre otros.

Normalmente, el atacante busca aparatos de marcas no muy conocidas, porque al no tener suficiente mano de obra, probablemente tengan vulnerabilidades


Cuando al atacante encuentra un aparato, intenta conectarse con credenciales predeterminadas, que normalmente la gente no suele cambiar.

Una vez que logra conectarse, el atacante manda un archivo al aparato, y ese Malware se replica, y lo manda a todos los dispositivos conectados

Ese Malware se va replicando a si mismo, y haci es que se forman las Botnets


El atacante, desde un servidor llamado Command & Control, o C&C, manda a todos esos aparatos a atacar el servidor.

Al tener tanta gente conectada, y el servidor no tener los recursos suficientes, el servidor se cae.



- 3. ¿Cómo instalar una distribución?

Para instalar una distribución de Linux, debes ingresar a la página de la distribución y descargar el archivo .iso Una vez descargado, deberas meter la ISO en un USB o en un CD.


En este caso, como estamos iniciando en Linux, y vamos a desarrollar, es recomendable instalar Ubuntu.

Para descargar Ubuntu, Presiona aquí


Una vez descargada la ISO, vamos a meterla en un USB o en un CD.

Para meter la ISO en un USB, vamos a usar el programa de Rufus


Una vez que tenemos la ISO en el USB, vamos a apagar el computador. Cuando la vayamos a prender de nuevo,

debemos iniciar desde la BIOS con el USB. Una vez que arranquemos con el USB, se nos abrira el proceso de instalación de Ubuntu.


El proceso de instalación es muy intuitivo, por lo que no tendrás problemas con la instalación.


Dualboot

Si tienes Windows en tu computador, y quieres instalar alguna distribución de Linux sin perder Windows, mira esto.


Al instalar cualquier distribución de Linux, podemos configurar el Dualboot,

lo que quiere decir que cuando prendamos el ordenador, podremos seleccionar si iniciar con Windows o con la distribución de Linux.


Para esto, debemos de crear una nueva partición, y seleccionar la partición donde vamos a instalar la distribución en este caso, Ubuntu.

Al iniciar el computador, nos saldrá la opción de iniciar con Windows o con la distribución, en este caso, Ubuntu.
