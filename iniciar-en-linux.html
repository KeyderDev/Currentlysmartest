Inicio del Curso

- 1. Que es linux?

Linux es un sistema operativo de codigo abierto donde

cualquier persona puede modificarlo y hacer sus propias versiones.


Si aprendes, tu mismo puedes hacer tu propia distribucion de Linux.



- 2. Que son las distros de Linux?

Las distribuciones de Linux son Sistemas Operativos basados en este,

es decir, son sistemas operativos hechos por la comunidad apartir de Linux.


Las distribuciones mas famosas son:


- Ubuntu

- Arch Linux

- Fedora

- SUSE Linux

- Entre otras...



- 3. Instalacion de una distro

Para instalar una distribucion de Linux, debes ingresar a la pagina de la distro y descargar

el archivo .iso Una vez descargado, deberas meter la iso en un USB o en un CD.


En este caso, como estamos iniciando en Linux, y vamos a desarrollar, es recomendable instalar Ubuntu.

Para descargar Ubuntu, Haz click aqui


Una vez descargada la ISO, vamos a meterla en un USB o en un CD.

Para meter la ISO en un USB, vamos a usar el programa de Rufus


Una vez que tenemos la ISO en el USB, vamos a apagar la PC. Cuando la vayamos a prender de nuevo,

debemos iniciar desde la BIOS con el USB. Una vez que arranquemos con el USB, se nos abrira el proceso de instalacion de Ubuntu.


El proceso de instalacion es muy intuitivo, por lo que no tendras problemas en la instalacion.


Dualboot

Si tienes Windows en tu PC, y quieres instalar alguna distro de Linux sin perder Windows, mira esto.


Al instalar cualquier distro de Linux, podemos configurar el Dualboot, lo que quiere decir que cuando

prendamos el ordenador, podremos seleccionar si iniciar con Windows o con la distro.
